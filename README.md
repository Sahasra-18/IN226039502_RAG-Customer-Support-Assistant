# RAG-Based Customer Support Assistant

## Overview
This project implements a Retrieval-Augmented Generation (RAG) system using LangChain and LangGraph.

## Features
- PDF-based knowledge retrieval
- Embeddings using HuggingFace
- ChromaDB vector storage
- LangGraph workflow
- HITL (Human-in-the-Loop)

## Tech Stack
- Python
- LangChain
- LangGraph
- ChromaDB

## How it works
1. Load PDF
2. Split into chunks
3. Create embeddings
4. Store in vector DB
5. Retrieve relevant data
6. Generate response

- main.ipynb
- Customer_Support_Knowledge_Base.pdf
